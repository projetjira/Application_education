Service de Favoris

Bienvenue sur l'API du service de gestion des favoris.

Documentation de l'API

Pour accéder à la documentation complète de l'API avec Swagger UI :

/swagger-ui/index.html

Points d'accès principaux

  • Récupérer les favoris d'un utilisateur : /api/favorites/user/{userId}
  • Récupérer les favoris par type : /api/favorites/user/{userId}/type/{contentType}
  • Ajouter un favori : POST /api/favorites
  • Supprimer un favori : DELETE /api/favorites/user/{userId}/content/{contentId}
  • Basculer l'état d'un favori : POST /api/favorites/toggle
